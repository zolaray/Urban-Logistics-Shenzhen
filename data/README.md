# Shenzhen Traffic Datasets

This folder contains the metadata for the spatio-temporal analysis:

1. **SZ-taxi Dataset:** - Origin: Shenzhen Taxi Trajectory Data.
   - Size: 156 road segments (nodes).
   - Features: Traffic speed sampled every 15 minutes.

2. **Road Topology:** - Extracted from OpenStreetMap (OSM) using OSMnx.
   - Focused on the **Luohu District** (Major logistics hub in Shenzhen).
